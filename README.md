# JLPT Kanji Study Site

Энэ репозиторид JLPT N5 → N1 түвшний канжи судлах интерактив вэб сайт байна.  

Онцлог:
- Канжи жагсаалт, уншлага, утга
- Accordion/Collapse функцийг ашиглах
- Flashcard / Quiz модуль
- Dark mode
- Responsive layout (гар утас, таблет, компьютер)

### Хэрэглэх заавар
1. Репозиторыг GitHub Pages-д байрлуулж вэб сайт болгон ашиглах
2. `index.html` файл root-д байх ёстой
3. Вэб хөтөч (Chrome, Edge, Firefox, Safari)-оор нээж үзнэ
